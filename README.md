# Fertility Timing and Household Decision-Making: An Econometric Study

Econometric study examining the effect of female household headship on fertility timing in Taiwan, using Post-Double Selection Lasso across 100+ covariates and OLS with cohort fixed effects.

## Overview
- Analyzed Taiwan's national household survey data (n=13,856) to examine intra-household power and reproductive behavior
- Applied PDS-Lasso for variable selection to minimize omitted variable bias in a high-dimensional covariate space
- Conducted robustness checks across three model specifications, finding a consistent negative effect of female headship on age at first birth
